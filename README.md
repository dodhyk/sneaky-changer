# Sneaky Changer (Ultimate Prank) 😈😇

**The most "annoying" yet "safest" prank for your coding buddies.**

Ever wanted to see your friend struggle with invisible syntax errors while keeping their repository perfectly clean? **Sneaky Changer** is here to help!

## 🚀 Features

- **Hard Mode (Evil)**: Automatically turns semicolons (`;`) into Greek Question Marks (`;`) in the editor and on disk. This causes **Build Errors** in the terminal that look identical to normal code!
- **Kind-Hearted Safety**: Automatically injects a hidden Git Hook that fixes everything back to normal semicolons before a `git commit`. 
- **Secret Toggle**: A cryptic status bar item that only you know how to use.
- **Auto-Activation**: Starts working silently in the background.

## 🛠 How to Use

1. Install the extension on your friend's VS Code.
2. Look for the small icon in the **Status Bar** (bottom right).
3. Click it to toggle **Sneaky Mode (HARD)**.
4. Watch the red squiggles and build errors drive them crazy!
5. Relax knowing that their Git history will remain 100% clean.
